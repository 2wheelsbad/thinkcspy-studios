# Studio: Holiday

## Concepts used

- Input (`input`)
- Type conversion
- Variable assignment
- Output (`print`)
- Modulo

## Walkthrough

Create a tip calculator. The program should prompt the user for the subtotal for their meal, along with the tip percentage, and print out the calculated tip.
